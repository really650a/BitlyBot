# BitlyBot
Bitly Short link generator Bot 


## Configs

* TOKEN  - Get bot token from @BotFather

* API_ID     - From my.telegram.org 

* API_HASH    - From my.telegram.org

* BITLY_TOKEN  - From https://bitly.is/accesstoken
